# AGPL-3.0 License Example

This package is an example of software using the GNU Affero General Public License v3.0 (AGPL-3.0). It's designed to be detected by Software Composition Analysis (SCA) tools as containing a forbidden license.

## Important License Information

This package is licensed under the GNU Affero General Public License v3.0 (AGPL-3.0).

Key restrictions of AGPL-3.0:
- Must disclose source code when distributing
- Must include original license
- Must state changes made to the code
- Network use is considered distribution
- Provides patent rights

## Dependencies

This package depends on MongoDB which is licensed under the Server Side Public License (SSPL), which is effectively equivalent to AGPL-3.0 for most use cases.

## Purpose

This package exists solely to demonstrate how SCA tools detect and flag AGPL-3.0 licensed software in a codebase.
